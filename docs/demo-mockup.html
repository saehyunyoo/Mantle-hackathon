●HOW MINTLY WORKS

매일 09:00 UTC, 자율 루프가 돈다.

01
Discover

NASDAQ·KOSPI 거래량 + 옵션 IV + 외국인 매수 + 소셜 시그널을 자동 스캔.

02
Decide

에이전트가 자산·형태(spot · 3× · index)·venue를 자율 결정. 추론은 비공개.

03
Mint on Mantle

DailyTokenFactory가 ERC-20 발행, LPRouter가 venue에 자동 시드.

04
Verify

24시간 후 LP fee · IL · 가격 추적 정확도로 자동 채점. 사람이 끼지 않는다.

05
Reputation

결과가 ERC-8004 NFT에 영구 기록. 평판은 시간이 지날수록 자산이 된다.

알파는 노출되지 않는다.

결정·결과·평판은 공개. 추론 알고리즘은 빌더 서버 안에 있다. 빌더가 4가지 모드 중 선택한다.

A
Open Source
전체 공개 · 학습용
B
Decisions Only · DEFAULT
결정·결과만 공개. 추론 비공개.
C
Time-Delayed
결정도 24h 후 공개. 알파 보호.
D
ZK Mode
증명만 온체인. 헤지펀드 그레이드.

왜 이게 베껴지지 않나.

코드는 베껴도 시간은 못 베낀다. 4가지 모트.

1 · 평판 NFT의 누적 시간
30·90·365일 트랙레코드 자체가 자산. 후발주자는 처음부터 쌓아야 한다.
2 · 빌더 네트워크 락인
빌더의 평판 NFT가 우리 컨트랙트에 있다. 옮기려면 0에서 시작.
3 · 추천 vs 결과 데이터셋
매일 누적되는 채점 데이터 자체가 운용사·연구기관에 팔린다.
4 · Mantle deep integration
생태계 합의 + 자산 라우팅 우선권 = 분배 채널 모트.
